HTML

Trata-se de uma linguagem de marcação de hipertexto, recurso utilizado para desenvolver páginas Web para levar informação a qualquer pessoa no mundo, independente da distância física.

A linguagem foi criada por Tim Beeners-Lee, o físico e cientista da computação britânico que também criou a World Wide Web, uma das principais infraestruturas que compõem a internet atual.
